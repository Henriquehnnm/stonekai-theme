## About The Theme

Stonekai is a theme that provides a cool and modern feel for your Visual Studio Code environment. It aims to be the definitive modern, unofficial version of the classic Monokai theme. It was designed to be easy on the eyes, while maintaining a high level of readability and contrast.

## Installation

1.  Open **Visual Studio Code**.
2.  Go to the **Extensions** view (`Ctrl+Shift+X`).
3.  Search for `Stonekai Theme`.
4.  Click **Install** to apply the theme.
5.  Go to **File** > **Preferences** > **Color Theme** and select `Stonekai`.

Enjoy!
